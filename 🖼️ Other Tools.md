这些项目涉及区块链数据处理、挖矿和其他功能。

- **[manuscript-core]** 📊  
  区块链数据流框架，整合链上和链下数据。  
  链接: https://github.com/chainbase-labs/manuscript-core
- **[cryo]** 📊  
  提取区块链数据到 parquet、csv、json 或 Python 数据框架。  
  链接: https://github.com/paradigmxyz/cryo
- **[cryo]** 📊  
  提取区块链数据工具（另一版本）。  
  链接: https://github.com/fuzzland/cryo
- **[ore-miner]** ⛏️  
  基于 Jito 捆绑的 ORE 挖矿工具，支持 CPU 和 GPU。  
  链接: https://github.com/tonyke-bot/ore-miner
- **[atomicals-miner]** ⛏️  
  Atomicals 挖矿优化版本，持续更新。  
  链接: https://github.com/neal-zhu/atomicals-miner
- **[collider]** ⛏️
