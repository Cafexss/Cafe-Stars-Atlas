这些项目涉及 AI 模型、自动化代理和相关框架。

- **[new-api]** 🧠  
  AI 模型接口管理和分发系统，支持多种大模型。  
  链接: https://github.com/QuantumNous/new-api
- **[swarms]** 🤖  
  企业级多代理协调框架。  
  链接: https://github.com/kyegomez/swarms
- **[solana-agent-kit]** 🤖  
  将 AI 代理连接到 Solana 协议的工具包。  
  链接: https://github.com/sendaifun/solana-agent-kit
- **[AI-Investigator]** 🧠  
  使用 Claude 3.5 Sonnet 和 Firecrawl 分析网站内容的 AI 系统。  
  链接: https://github.com/muratcankoylan/AI-Investigator
- **[cerebellum]** 🌐  
  使用 AI 驱动的浏览器自动化系统，执行网页导航任务。  
  链接: https://github.com/theredsix/cerebellum
- **[comfyui_LLM_party]** 🧠  
  ComfyUI 中的 LLM 代理框架，支持多种模型和功能。  
  链接: https://github.com/heshengtao/comfyui_LLM_party
- **[chatgpt-on-wechat]** 🤖  
  基于大模型的聊天机器人，支持多平台接入。  
  链接: https://github.com/zhayujie/chatgpt-on-wechat
- **[bilibot]** 🤖  
  使用 Bilibili 用户评论微调的本地聊天机器人。  
  链接: https://github.com/linyiLYi/bilibot
- **[mlx]** 🧠  
  面向 Apple 芯片的数组框架。  
  链接: https://github.com/ml-explore/mlx
- **[timesfm]** 📊  
  Google 研究的预训练时间序列基础模型。  
  链接: https://github.com/google-research/timesfm
- **[dive-into-llms]** 📚  
  大模型编程实践教程。  
  链接: https://github.com/Lordog/dive-into-llms
- **[ChatTTS]** 🗣️  
  用于日常对话的生成式语音模型。  
  链接: https://github.com/2noise/ChatTTS
- **[Thinking-Claude]** 🧠  
  增强 Claude 的思考能力。  
  链接: https://github.com/richards199999/Thinking-Claude
- **[wiseflow]** 🧠  
  使用 LLM 从海量信息中提取关键内容的工具。  
  链接: https://github.com/TeamWiseFlow/wiseflow
- **[openai-realtime-api-beta]** 🗣️  
  OpenAI 实时 API 的 Node.js 和 JavaScript 参考客户端。  
  链接: https://github.com/openai/openai-realtime-api-beta
- **[generative_agents]** 🧠  
  交互式人类行为模拟的生成代理。  
  链接: https://github.com/joonspk-research/generative_agents
- **[openai-translator]** 🌐  
  基于 ChatGPT API 的划词翻译插件和桌面应用。  
  链接: https://github.com/openai-translator/openai-translator
- **[myGPTReader]** 🤖  
  社区驱动的 AI 聊天阅读工具，基于 ChatGPT。  
  链接: https://github.com/myreader-io/myGPTReader
- **[bob-plugin-openai-translator]** 🌐  
  基于 OpenAI API 的 Bob 翻译插件。  
  链接: https://github.com/openai-translator/bob-plugin-openai-translator
- **[gpt-index-extension]** 🌐  
  Google Chrome 的 GPT 索引扩展。  
  链接: https://github.com/hongyishi/gpt-index-extension
- **[chatgpt-google-extension]** 🌐  
  已废弃的 ChatGPT Google 扩展，推荐新项目 ChatHub。  
  链接: https://github.com/wong2/chatgpt-google-extension
