这些项目专注于网络爬虫、数据抓取和社交媒体监控。

- **[cloudflare-scrape]** 🌐  
  Python 模块，绕过 Cloudflare 的反机器人页面。  
  链接: https://github.com/Anorov/cloudflare-scrape
- **[twint]** 📱  
  高级 Twitter 抓取和 OSINT 工具，无需 API 限制。  
  链接: https://github.com/twintproject/twint
- **[twikit]** 📱  
  Twitter API 抓取工具，无需 API 密钥。  
  链接: https://github.com/d60/twikit
- **[nitter]** 🌐  
  替代 Twitter 前端。  
  链接: https://github.com/zedeus/nitter
- **[twitter-monitor]** 📱  
  Twitter 抓取核心和相关应用。  
  链接: https://github.com/BANKA2017/twitter-monitor
- **[twspace-crawler]** 📱  
  24/7 监控和下载 Twitter Spaces 的脚本。  
  链接: https://github.com/HitomaruKonpaku/twspace-crawler
- **[agent-twitter-client]** 📱  
  无需 API 密钥的 Twitter 客户端，适合代理使用。  
  链接: https://github.com/elizaOS/agent-twitter-client
- **[x-crawl]** 🌐  
  灵活的 Node.js AI 辅助爬虫库。  
  链接: https://github.com/coder-hxl/x-crawl
- **[crawlee]** 🌐  
  Node.js 爬虫和浏览器自动化库，支持多种格式下载。  
  链接: https://github.com/apify/crawlee
- **[WeReadScan]** 📚  
  扫描微信读书已购图书并下载 PDF 的爬虫。  
  链接: https://github.com/Algebra-FUN/WeReadScan
- **[CyberScraper-2077]** 🌐  
  使用 LLM（OpenAI、Gemini、Ollama）驱动的强大网络爬虫。  
  链接: https://github.com/itsOwen/CyberScraper-2077
- **[Scraperr]** 🌐  
  自托管网络爬虫。  
  链接: https://github.com/jaypyles/Scraperr
- **[gecco]** 🌐  
  易用的轻量级 Java 网络爬虫。  
  链接: https://github.com/xtuhcy/gecco
- **[phpspider]** 🌐  
  PHP 爬虫，用于知乎百万用户数据抓取。  
  链接: https://github.com/owner888/phpspider
- **[weibo-public-opinion-analysis]** 📊  
  基于微博的舆情分析，包括爬虫和情感分析。  
  链接: https://github.com/stay-leave/weibo-public-opinion-analysis
- **[DNSLog]** 📡  
  监控 DNS 解析和 HTTP 访问记录的工具。  
  链接: https://github.com/BugScanTeam/DNSLog
